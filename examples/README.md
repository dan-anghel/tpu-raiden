# Raiden Examples

## Overview
This `examples/` folder contains scripts and microbenchmarks demonstrating how to interact with and evaluate TPU Raiden.

## How to Run
For specific execution details, prerequisites, and instructions on how to interpret outputs, please refer to the README inside each specific case directory:

* [Microbenchmarks](microbenchmarks/README.md): Microbenchmarks designed to test the raw DMA bandwidth (Host-to-Device and Device-to-Host) of the Raiden engine without Python or framework-level overhead.
